criar serializer:
entrar no app correspondente, criar arquivo serializers.py	
importar :
from rest_framework import serializers
from appName.models import ModelName

**Para usar um serializer eu também preciso configurar o na class Meta, model e fields**

criar class de serializers:
class ModelNameSerializer(serializers.ModelSerializer):
  class Meta:
     model = ModelName
     fields = '__all__' (posso listar os campos que eu quero) - exemplo ('id', 'name')


Depois de criar o serializer, abrir arquivos views e importar o serializer
from appName.serializers import serializerName
